# 統計分析方法學習指南 (Statistical Methods Guide)

本文件旨在為非統計背景的讀者提供本專案所使用之統計方法的入門介紹。內容包含方法的**基本概念**、**使用時機**、**關鍵指標解讀**，以及**在本專案中的具體應用**。

---

## 1. 皮爾森積差相關係數 (Pearson Correlation Coefficient)

### 1.1 基本概念

這是一種用來衡量兩個連續變數之間「線性關係」強度的指標。它的數值範圍在 -1 到 +1 之間。

- **+1**：完全正相關（一個變數增加，另一個也等比例增加）。
- **0**：沒有線性關係。
- **-1**：完全負相關（一個變數增加，另一個等比例減少）。

### 1.2 使用時機

當你想知道「變數 A 變大時，變數 B 是不是也會跟著變大或變小？」的時候使用。
_注意：相關不代表因果。A 與 B 相關，不代表是 A 導致了 B。_

### 1.3 關鍵指標

- **相關係數 (r)**：
  - $|r| > 0.7$：高度相關
  - $0.3 < |r| < 0.7$：中度相關
  - $|r| < 0.3$：低度相關
- **p 值 (p-value)**：檢定這個相關性是否顯著（是否只是剛好抽樣抽到的）。通常要求 p < 0.05。

### 1.4 本專案應用

- **分析對象**：「出生人口數」 vs. 「統測總報名人數」。
- **分析結果**：$r = 0.97$ (高度正相關)，$p < 0.01$。
- **解讀**：出生人口越少，報名人數就越少。這證實了「少子化」是導致學生減少的根本原因（推力）。

## 2. 單因子變異數分析 (One-way ANOVA)

### 2.1 基本概念

用來比較**三個或以上**組別的平均數是否有差異的統計方法。
它透過比較「組間差異」（不同組別的差別）與「組內差異」（同一組別內的波動）來判斷。如果組間差異遠大於組內差異，就代表這些組別真的不同。

### 2.2 使用時機

當你有一個類別變數（如：行業別）和一個連續變數（如：薪資），且類別超過兩組時。
_例如：比較「金融業」、「餐飲業」、「製造業」三種行業的平均薪資是否一樣。_

### 2.3 關鍵指標

- **F 值 (F-statistic)**：組間差異除以組內差異的比值。F 值越大，代表組別間的差異越顯著。
- **p 值**：若 p < 0.05，代表至少有一組的平均數跟其他組不一樣。

### 2.4 本專案應用

- **分析對象**：不同「行業別」的「平均總薪資」。
- **分析結果**：$F = 233.99$， $p < 0.001$。
- **解讀**：不同行業的薪資有極顯著的差異。這證明了「薪資」在市場上是一個可以區分不同職業的有效指標（例如金融業薪資顯著高於餐飲業）。

## 3. 簡單線性迴歸 (Simple Linear Regression)

### 3.1 基本概念

建立一個數學方程式 ($Y = \beta_0 + \beta_1 X$)，用一個自變數 ($X$) 來預測一個應變數 ($Y$)。

- **截距 ($\beta_0$)**：當 $X=0$ 時，$Y$ 的預測值。
- **斜率 ($\beta_1$)**：$X$ 每增加 1 單位，$Y$ 預期會增加或減少的量。

### 3.2 使用時機

當你想量化 $X$ 對 $Y$ 的影響程度，或者想用 $X$ 來預測 $Y$ 時。
_例如：用「廣告費用」來預測「銷售額」。_

### 3.3 關鍵指標

- **判定係數 ($R^2$)**：模型能解釋 $Y$ 變異的百分比。越高代表模型越準。
- **係數顯著性 (p-value)**：檢定斜率是否顯著不為 0。若 p > 0.05，代表 $X$ 對 $Y$ 沒有顯著影響。

### 3.4 本專案應用

- **分析對象**：用「薪資年增率」($X$) 預測「報名人數年增率」($Y$)。
- **分析結果**：$p = 0.915$ (不顯著)，係數接近 0。
- **解讀**：薪資成長率無法預測報名人數的變化。這推翻了「加薪就能招到更多學生」的直觀假設，引導出「薪資是護城河（抗跌）」而非「推進器（成長）」的深層結論。

## 4. 視覺化分析：散佈圖與趨勢線 (Scatter Plot & Trend Line)

### 4.1 基本概念

- **散佈圖**：將每個資料點畫在二維平面上，直觀展示兩個變數的分佈型態。
- **趨勢線**：畫出一條最能代表資料走向的直線（通常使用最小平方法 OLS）。

### 4.2 使用時機

在進行複雜統計前，先用眼睛觀察資料的「長相」。可以發現線性關係、非線性關係、或離群值。

### 4.3 本專案應用

- **圖表**：決策矩陣圖（X 軸：薪資年增率，Y 軸：報名人數年增率）。
- **觀察**：雖然視覺上好像高薪群類衰退較少，但畫出趨勢線後發現斜率幾乎是平的。
- **解讀**：這提醒我們**「眼見不一定為憑」**。視覺上的群聚（高薪群類在上方，低薪群類在下方）是存在的，但整體線性的「連動關係」在統計上並不成立。這幫助我們區分「存量（薪資水準）」與「流量（薪資增長）」的不同影響。
