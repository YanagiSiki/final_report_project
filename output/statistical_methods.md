# 統計分析方法學習指南 (Statistical Methods Guide)

本文件旨在為非統計背景的讀者提供本專案所使用之統計方法的入門介紹。內容包含方法的**基本概念**、**使用時機**、**關鍵指標解讀**，以及**在本專案中的具體應用**。

---

## 1. 皮爾森積差相關係數 (Pearson Correlation Coefficient)

### 1.1 基本概念

這是一種用來衡量兩個連續變數之間「線性關係」強度的指標。它的數值範圍在 -1 到 +1 之間。

- **+1**：完全正相關（一個變數增加，另一個也等比例增加）。
- **0**：沒有線性關係。
- **-1**：完全負相關（一個變數增加，另一個等比例減少）。

### 1.2 使用時機

適用於檢驗兩個連續變數間是否存在線性關聯性。
_注意：相關性不代表因果關係。_

### 1.3 關鍵指標

- **相關係數 (r)**：
  - $|r| > 0.7$：高度相關
  - $0.3 < |r| < 0.7$：中度相關
  - $|r| < 0.3$：低度相關
- **p 值 (p-value)**：檢定這個相關性是否顯著（是否只是剛好抽樣抽到的）。通常要求 p < 0.05。

### 1.4 本專案應用

- **分析對象**：「出生人口數」與「統測總報名人數」之間的關聯。
- **分析結果**：$r = 0.97$ (高度正相關)，$p < 0.01$。
- **解讀**：出生人口減少與報名人數減少呈現高度正相關，顯示「少子化」為學生人數下降的主要背景因素。

## 2. 單因子變異數分析 (One-way ANOVA)

### 2.1 基本概念

用於比較三個或以上組別的平均數是否存在顯著差異。其原理為比較「組間差異」與「組內差異」的相對大小，若組間差異顯著大於組內差異，則可推論組別間存在顯著差異。

### 2.2 使用時機

適用於一個類別變數（如行業別）與一個連續變數（如薪資），且類別數超過兩組時。
_例如：比較多個行業的平均薪資是否存在顯著差異。_

### 2.3 關鍵指標

- **F 值 (F-statistic)**：組間差異除以組內差異的比值。F 值越大，代表組別間的差異越顯著。
- **p 值**：若 p < 0.05，代表至少有一組的平均數跟其他組不一樣。

### 2.4 本專案應用

- **分析對象**：不同「行業別」的「平均總薪資」。
- **分析結果**：$F = 233.99$， $p < 0.001$。
- **解讀**：不同行業的薪資存在極顯著差異，顯示「薪資」為區分行業特性的有效指標。

## 3. 簡單線性迴歸 (Simple Linear Regression)

### 3.1 基本概念

建立一個數學方程式 ($Y = \beta_0 + \beta_1 X$)，以一個自變數 ($X$) 預測一個應變數 ($Y$)。

- **截距 ($\beta_0$)**：當 $X=0$ 時，$Y$ 的預測值。
- **斜率 ($\beta_1$)**：$X$ 每增加 1 單位，$Y$ 預期變化的量。

### 3.2 使用時機

適用於量化 $X$ 對 $Y$ 的影響程度，或以 $X$ 預測 $Y$。
_例如：以某變數預測另一變數的變化。_

### 3.3 關鍵指標

- **判定係數 ($R^2$)**：模型能解釋 $Y$ 變異的百分比。越高代表模型越準。
- **係數顯著性 (p-value)**：檢定斜率是否顯著不為 0。若 p > 0.05，代表 $X$ 對 $Y$ 沒有顯著影響。

### 3.4 本專案應用

- **分析對象**：以「薪資年增率」($X$) 預測「報名人數年增率」($Y$)。
- **分析結果**：$p = 0.915$ (不顯著)，係數接近 0。
- **解讀**：薪資成長率對報名人數變化無顯著預測力，顯示兩者間不具線性關聯。

## 4. 視覺化分析：散佈圖與趨勢線 (Scatter Plot & Trend Line)

### 4.1 基本概念

- **散佈圖**：將每個資料點畫在二維平面上，直觀展示兩個變數的分佈型態。
- **趨勢線**：畫出一條最能代表資料走向的直線（通常使用最小平方法 OLS）。

### 4.2 使用時機

適用於進行複雜統計分析前，先以視覺化方式檢查資料分布、線性或非線性關係及離群值。

### 4.3 本專案應用

- **圖表**：決策矩陣圖（X 軸：薪資年增率，Y 軸：報名人數年增率）。
- **觀察**：視覺上高薪群類衰退幅度較小，但趨勢線斜率接近於零。
- **解讀**：視覺上雖可觀察群聚現象，但整體線性關聯未達統計顯著，顯示需區分薪資水準與薪資增長對學生選擇的不同影響。

## 5. 市場佔比分析 (Market Share Analysis)

### 5.1 基本概念

將絕對數值（如人數）轉換為相對比例（佔比），公式為 $P_i = \frac{N_i}{\sum N}$。

### 5.2 使用時機

當總體規模（Total Market Size）發生劇烈變化時（如少子化導致總學生數下降），僅觀察人數易產生誤判。轉換為「佔比」可排除總體趨勢干擾，更精確反映各類別間的相對競爭力。

### 5.3 本專案應用

- **應用說明**：少子化導致所有群類人數下降，僅觀察人數難以辨識相對優勢。透過計算市佔率，可發現即使某群類人數減少，若其市佔率上升，則在縮小的市場中相對競爭力提升，有助於檢驗薪資結構對學生選擇偏好的影響。

## 6. 軌跡分析 (Trajectory Analysis)

### 6.1 基本概念

將時間序列數據在二維平面上繪製成路徑，以觀察變數隨時間演變的動態方向。通常 X 軸與 Y 軸分別代表兩個關鍵變數（如薪資與市佔率），點代表年份，箭頭指示時間流向。

### 6.2 使用時機

適用於觀察兩個變數的「共同演化」過程，特別是當單純的時間序列圖難以呈現兩者互動關係時。

### 6.3 本專案應用

- **圖表**：產業軌跡圖（X 軸：平均薪資，Y 軸：市場佔有率）。
- **觀察**：高薪群類（如電資）軌跡向右上方移動（薪資增、市佔增）；低薪群類（如餐旅）軌跡向左下方移動（薪資緩漲、市佔減）。
- **解讀**：直觀呈現了「吸納效應」，即高薪資如何驅動市佔率的長期擴張。

---

## 7. 3 年週期分析 (3-Year Cycle Analysis)

### 7.1 基本概念

將連續的時間序列數據重新取樣，以 3 年為一個計算單位（例如：計算 3 年間的總成長率），而非逐年計算。

### 7.2 使用時機

- **過濾雜訊**：當年度數據波動較大（如受疫情短期衝擊）時，3 年週期可平滑短期波動，凸顯長期趨勢。
- **配合學制**：技職教育為 3 年制，學生從入學到畢業報考統測存在時間滯後 (Time Lag)，3 年週期更能反映教育選擇的決策週期。

### 7.3 本專案應用

- **應用說明**：計算每 3 年的「薪資成長率」與「市佔率成長率」。
- **分析結果**：在過濾短期雜訊後，發現薪資成長與市佔率成長呈現更清晰的正向關聯。
- **解讀**：證實學生對薪資的反應具有「長期理性」，短期波動雖有干擾，但長期而言，資源確實向高成長產業集中。
