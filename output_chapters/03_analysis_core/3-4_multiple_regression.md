# 3.4 多元迴歸模型：探討人口與薪資的聯合影響 (Multiple Regression Model: Joint Impact of Population and Salary)

在前面的章節中，我們分別探討了少子化的「推力」以及行業薪資的「拉力」對學生科系選擇的影響。本節將建立一個多元迴歸模型，旨在量化這些因素對於科系報名人數的聯合解釋力，以驗證本研究的核心假說：在控制少子化這個宏觀背景後，高薪產業對應的科系是否能更有效地抵抗生源流失。

---

### 3.4.1 分析方法

我們使用多元線性迴歸模型來預測各科系群類的報名人數。

-   **應變數 (Dependent Variable, Y)**: 各科系群類的歷年報名人數。
-   **自變數 (Independent Variables, $X_1$, $X_2$)**:
    -   $X_1$: 對應年度的出生總人口 (已標準化)。
    -   $X_2$: 對應行業的歷年平均薪資 (已標準化)。

模型公式設定為：$Y (\text{報名人數}) = \beta_0 + \beta_1(\text{出生人數\_scaled}) + \beta_2(\text{薪資\_scaled}) + \epsilon$

本模型期望檢視：
1.  $\beta_1$ (出生人數\_scaled) 的顯著性與係數：預期出生人數越多，報名人數越多。
2.  $\beta_2$ (薪資\_scaled) 的顯著性與係數：預期在控制出生人數後，薪資越高，報名人數越多（或流失越慢）。

### 3.4.2 分析結果與討論

多元迴歸模型的結果摘要如下：


**結果判讀：**

1.  **模型整體顯著性**：
    -   模型的 F 統計量為 1.32，對應的 p 值為 0.2813。由於 p 值大於傳統的 0.05 顯著水準，這表示**該多元迴歸模型整體不具備統計顯著性**。換言之，出生人數和行業薪資這兩個變數，並未能聯合顯著地解釋科系報名人數的變異。

2.  **自變數的個別影響**：
    -   **出生人數 (出生人數\_scaled)**：其迴歸係數為 1166.0，p 值為 0.241。此結果**不具統計顯著性**。這與我們原先預期出生人數會對報名人數產生顯著影響的假說不符。在控制薪資後，出生人數的獨立影響在此模型中未達統計顯著。
    -   **行業薪資 (薪資\_scaled)**：其迴歸係數為 1216.4，p 值為 0.222。此結果亦**不具統計顯著性**。這是一個關鍵發現，它表明**在控制出生人數變化的影響後，行業的平均薪資水平並未顯著影響科系的報名人數**。這與本研究的核心假設——學生選擇科系具有「經濟理性」（即高薪科系流失較慢）——相悖。

3.  **模型解釋力**：
    -   Multiple R-squared 為 0.07622，調整後 R-squared 僅為 0.01848。這表示模型解釋了報名人數變異的約 1.85%，是一個**非常低的解釋力**。

### 3.4.3 小結

多元迴歸分析的結果顯示，儘管在描述性統計和視覺化分析中，我們觀察到薪資與報名人數變動趨勢之間存在一些關聯性，但在這個多元迴歸模型中，**出生人數和行業平均薪資這兩個自變數，均未能顯著預測科系的報名人數**。模型的整體解釋力也極其有限。

這暗示了影響學生科系選擇的因素可能更為複雜，單純的出生人口數和行業平均薪資（或其線性組合）不足以全面解釋報名人數的變動。除了經濟因素外，學生的興趣、社會價值觀、科系聲譽、教育政策、甚至非經濟性的工作環境考量（如勞動強度、工時等）可能扮演了更重要的角色，或者模型中存在其他未納入的關鍵干擾變數。儘管本模型未能支持核心假說，但這本身也是一個重要的研究發現，將為未來的研究提供新的方向。
