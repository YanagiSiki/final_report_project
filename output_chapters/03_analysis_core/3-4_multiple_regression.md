# 3.4 進階迴歸模型：檢驗增長率的動態關係 (Advanced Regression Model: Examining the Dynamic Relationship of Growth Rates)

在 3.3 節的視覺化分析中，我們觀察到「薪資年增率」與「報名人數年增率」之間似乎存在一個正向的趨勢。為了用更嚴謹的統計方法來檢驗這個視覺觀察，本節將建立一個以「增長率」為核心變數的進階迴歸模型。

---

### 3.4.1 分析方法

本節採用**簡單線性迴歸模型 (Simple Linear Regression)**。此方法透過普通最小二乘法 (OLS)，找出最能適配資料點的一條直線，以估計一個自變數 (X) 對一個應變數 (Y) 的線性影響。我們的目標是量化「薪資年增率」對「報名人數年增率」的預測能力，並檢定此關係的統計顯著性。

-   **迴歸模型公式**:
    $$
    Y_i = \beta_0 + \beta_1 X_i + \epsilon_i
    $$

-   **變數說明**:
    -   **應變數 ($Y_i$)**: 第 $i$ 個觀察樣本的「報名人數年增率」。
    -   **自變數 ($X_i$)**: 第 $i$ 個觀察樣本的「薪資年增率」。

-   **係數說明**:
    -   **截距項 ($\beta_0$)**: 代表當「薪資年增率」為 0% 時，模型所預測的「報名人數年增率」的平均值。它反映了在沒有薪資增長影響下的基礎變動趨勢。
    -   **斜率項 ($\beta_1$)**: 此為本研究最關注的係數。它代表「薪資年增率」每增加一個單位（即 1%），「報名人數年增率」預期會改變的量。若此係數為顯著的正數，則支持我們的核心假說。

本模型所檢驗的變數關係，已於 3.3 節**圖 3.3.1** 中視覺化呈現。本節旨在提供對該視覺趨勢的量化統計檢定。

### 3.4.2 分析結果與討論

迴歸模型的真實結果摘要如下：

```
Call:
lm(formula = 報名人數年增率 ~ 薪資年增率, data = analysis_data_for_regression)

Residuals:
      Min        1Q    Median        3Q       Max
-0.116129 -0.034588 -0.007508  0.031446  0.184636

Coefficients:
            Estimate Std. Error t value Pr(>|t|)
(Intercept) -0.06786    0.04444  -1.527    0.136
薪資年增率  -0.14605    1.35537  -0.108    0.915

Residual standard error: 0.06198 on 34 degrees of freedom
Multiple R-squared:  0.0003414, Adjusted R-squared:  -0.02906
F-statistic: 0.01161 on 1 and 34 DF,  p-value: 0.9148
```

**結果判讀：**

1.  **模型整體顯著性**：
    -   模型的 F 統計量僅為 0.01161，對應的 p 值高達 0.9148。這表示**該迴歸模型整體完全不具備統計顯著性**。

2.  **自變數的影響**：
    -   **薪資年增率**: 其 p 值為 0.915，遠大於 0.05 的顯著水準。這是一個決定性的發現：**「薪資年增率」對於「報名人數年增率」不具有任何顯著的預測能力**。
    -   更值得注意的是，其係數估計值為 **-0.146 (負數)**，這與我們在 3.3 節散佈圖中觀察到的視覺趨勢（正斜率）**完全相反**。

3.  **模型解釋力**：
    -   Adjusted R-squared 為 -0.02906，一個趨近於零的負值。這表示模型**完全無法解釋**「報名人數年增率」的任何變異。

### 3.4.3 小結

這個以增長率為基礎的進階迴歸模型，其結果並未支持本研究的核心假說，反而提供了一個更深刻的洞見：**視覺上的趨勢可能是誤導性的**。

儘管 3.3 節的散佈圖看似呈現正向關聯，但嚴謹的統計檢定顯示此關聯性並不穩定，甚至方向相反，且完全不具顯著性。這強烈暗示，至少在本研究的資料範圍與模型設定下，「薪資增長」並非影響「報名人數變化」的直接、線性驅動因素。學生的選擇行為，其背後的驅動因子遠比單純的薪資變化更為複雜，這也印證了我們在第一個迴歸模型中得到的初步結論。
