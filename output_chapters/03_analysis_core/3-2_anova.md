# 3.2 變異數分析：不同群體間的薪資比較 (ANOVA: Salary Comparison Across Groups)

在本節中，我們採用單因子變異數分析（One-Way ANOVA）來檢驗不同分類群體（例如：不同行業別）的平均總薪資是否存在統計上的顯著差異。這項分析有助於我們了解特定群體屬性是否為影響薪資水準的關鍵因素。

---

### 3.2.1 研究方法與假設

為了進行檢定，我們定義了以下變數與研究假設：

- **獨立變數 (Independent Variable):** 將資料依據「行業別」進行分組。
- **依變數 (Dependent Variable):** 各行業別對應的「平均總薪資」。

我們的研究假設如下：

- **虛無假設 (Null Hypothesis, H₀):**

  - 所有行業別的平均總薪資皆相等。換言之，行業別對薪資沒有顯著影響。

- **對立假設 (Alternative Hypothesis, H₁):**
  - 至少有一個行業別的平均總薪資與其他行業不相等。即行業別對薪資具有顯著影響。

我們將採用 **α = 0.05** 作為顯著水準。

### 3.2.3 事後檢定 (Post-Hoc Test)

當 ANOVA 結果呈現顯著 (p < 0.05) 時，我們可以確認至少有一組的平均值與其他組不同，但無法得知是哪些組之間存在差異。因此，我們需要進行事後檢定（如 **Tukey's HSD**）來進行兩兩比較。
