# 3.3 關鍵決策矩陣：薪資與招生率的視覺化分析 (Decision Matrix: Visualizing Salary and Enrollment Rates)

在分別確立了「少子化推力」與「產業薪資差異」後，本節旨在將兩者結合，透過一個關鍵的視覺化圖表——決策矩陣圖（散佈圖），直觀地呈現本研究的核心假說：行業薪資是否與科系招生率的消長存在關聯。

---

### 3.3.1 分析方法

我們繪製一個二維散佈圖，並以平均值為基準線，將圖表劃分為四個象限，以利進行策略性分析。
- **X 軸**: **113 年各對應行業平均總薪資**。代表該行業的「經濟吸引力」。
- **Y 軸**: **各統測群類 109-114 學年度報名人數變動率**。代表該群類的「學生選擇趨勢」。

圖中的每一個點，都代表一個「統測群類」及其對應的「行業薪資」。我們預期，如果學生的選擇是經濟理性的，那麼這些點應該會呈現由左下至右上的分佈趨勢。

### 3.3.2 圖表分析與討論

![薪資與人數變動率決策矩陣圖](../images/3_3_decision_matrix.svg)
_圖 3.3.1：行業薪資 vs. 群類人數變動率之決策矩陣圖_

**結果判讀：**
如圖 3.3.1 所示，散佈圖中的資料點**呈現了清晰的左下至右上的正向趨勢**，初步驗證了我們的假說。迴歸線的斜率為正，表示行業平均薪資越高，其對應的科系群類人數衰退越緩慢（甚至可能逆勢抗跌）。我們可以將圖中的群類分為四個象限來討論：

1.  **第一象限 (右上): 高薪資、高成長/低衰退**
    - **代表群類**: `衛生與護理類`、`電機與電子群`。
    - **解讀**: 這些群類對應的行業（醫療保健、資通訊）薪資優渥，因此即使在少子化浪潮下，依然是學生的熱門選擇，人數衰退幅度最小，最具競爭力。

2.  **第二象限 (左上): 低薪資、高成長/低衰退 (警示區)**
    - **代表群類**: (此區無明顯群類)
    - **解讀**: 理論上，此區代表「叫好不叫座」，即薪資不高但因興趣等其他因素報名人數多。在本次分析中此現象不顯著。

3.  **第三象限 (左下): 低薪資、低成長/高衰退 (重災區)**
    - **代表群類**: `餐旅群`、`外語群`、`藝術群`。
    - **解讀**: 這是本研究的重災區。這些群類對應的行業（住宿餐飲、休閒服務）薪資偏低，直接導致其在招生市場上吸引力嚴重不足，人數流失最為慘重。**`餐旅群`** 正是此象限的典型代表。

4.  **第四象限 (右下): 高薪資、低成長/高衰退 (潛力區)**
    - **代表群類**: `土木與建築群`、`機械群`。
    - **解讀**: 這些群類對應的行業（營建業、製造業）薪資並不低，但人數流失率卻高於預期。這暗示了除了薪資外，可能存在其他**非經濟的負面因素**，例如工作環境辛苦、勞動強度大、社會印象等，影響了學生的選擇意願。

### 3.3.3 小結

本節的視覺化分析提供了支持核心假說的直觀證據。數據點的分佈趨勢表明，**行業薪資水平與對應科系的招生穩定度有著密不可分的正向關係**。餐旅群的困境與資電/護理群的堅挺，在圖中形成了強烈對比。這個發現，為下一節使用更嚴謹的統計模型進行量化驗證，奠定了堅實的基礎。
