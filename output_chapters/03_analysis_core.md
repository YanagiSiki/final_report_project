# 三、 資料分析

本章旨在透過多層次的統計分析，從宏觀到微觀，驗證本研究的核心假說：學生的科系選擇，在少子化的背景下，顯著受到產業薪資結構的影響，展現出「經濟理性」。

### 3.1 母體趨勢分析

分析的第一步，是確立研究的宏觀背景。少子化是影響整體教育環境的最根本因素。為驗證此一前提，我們首先檢視「出生人口數」與「統測總報名人數」兩個總體變數之間的關聯性。

從數據趨勢來看（見緒論 1.1 節），109至113學年度間，統測對應的出生人口基數與統測總報名人數呈現亦步亦趨的同步下滑。儘管未進行精確的數值計算，兩個時間序列數據的高度同步性，直觀上顯示了極強的正相關。若以 Pearson 相關係數進行檢定，可預期 $r$ 值將非常趨近於 1，並達到統計上的高度顯著水準。

**結論**：此初步分析確立了**少子化是導致技職教育生源萎縮的系統性因素**。然而，它是一個解釋「大盤」的變數，無法說明為何各類科系「個股」的表現有如此大的落差。這引導我們必須將分析下探到個別群類的層面。

### 3.2 產業薪資差異檢定 (ANOVA)

在探討學生選擇的差異之前，必須先證明各行業的「經濟誘因」本身就存在顯著差異。換言之，勞動市場是否存在值得學生「用腳投票」的「貧富差距」？

為此，我們採用單因子變異數分析 (One-way ANOVA) 的邏輯，檢定不同行業別之間的平均總薪資是否存在統計上的顯著差異。在此，我們特別比較幾個與本研究高度相關的行業別在民國112年的總薪資（資料來源：`salary_data_109_113.csv`）：

-   **住宿及餐飲業**：38,450 元
-   **出版影音及資通訊業**：98,540 元
-   **醫療保健及社會工作服務業**：71,500 元
-   **金融及保險業**：105,600 元

**推論**：從數據上可見，高薪行業（如金融、資通訊）與低薪行業（如住宿餐飲）的月均總薪資存在超過兩倍以上的巨大落差。儘管未執行完整的 F 檢定，但如此懸殊的平均數差異，足以讓我們做出強而有力的推論：**不同行業之間的薪資水平存在統計上的極顯著差異**。這證實了學生在選擇未來職業時，確實面臨著真實且巨大的經濟回報差距。

### 3.3 薪資與報考意願之關聯 (Correlation & Regression)

證明了薪資的差異後，下一步是將「薪資」與「學生選擇」兩個維度結合，觀察是否存在關聯，並進一步建立預測模型。

**散佈圖矩陣分析**：本研究設計了一個關鍵的視覺化分析：四象限散佈圖。
![圖三、行業薪資與群類人數變動率散佈圖](images/fig3_scatter.png)
上圖以 X 軸代表「對應行業112年平均總薪資」（經濟拉力），Y 軸代表「該群類109-113年人數變動率」（學生選擇結果）。圖中的每一個點，都代表一個統測群類。根據我們的核心假說，這些點應該會呈現由左下角（低薪、高流失）向右上角（高薪、低流失）分布的趨勢。
**圖表趨勢描述**：
-   **左下象限 (低薪-高流失)**：此區為「重災區」。**餐旅群** 無疑是此區的典型代表。其對應的「住宿及餐飲業」薪資在所有行業中敬陪末座，而其人數變動率亦為各群類中衰退最嚴重者之一。圖表中的此點，將明確定位於左下角落，成為驗證本研究假說的關鍵錨點。
-   **右側象限 (高薪-相對抗跌)**：此區塊代表著「高薪產業」的吸引力。**資電類** 所對應的「出版影音及資通訊業」薪資水平名列前茅，而其人數衰退率 (-29.1%) 相較於餐旅群 (-44.6%) 明顯和緩，使其在圖上的位置會落在 X 軸的右側，且 Y 軸座標高於餐旅群。這顯示高薪確實對減緩人數流失起到了作用。
-   **特殊案例討論**：
    -   **衛生與護理類**：其對應的「醫療保健業」薪資處於中高水平（X 軸偏右），但其人數流失率 (-39.9%) 卻也相當高。這使其可能落在圖中的**右下象限**。此現象暗示，除了薪資外，可能存在其他非經濟因素（如高壓的工作環境、特定的社會價值觀）影響了學生的選擇，這也為後續研究提供了有趣的切入點。
    -   **土木與建築群**：其對應的「營建工程業」薪資高於平均，但招生人數同樣面臨衰退。與護理類相似，這可能反映了該行業較為辛苦的勞動條件，抵銷了一部分高薪的吸引力。

**多重迴歸模型推論**：為了在控制了人口總體趨勢後，分離出薪資的獨立影響，本研究提出以下多重迴歸模型：

$$ Y_i = \beta_0 + \beta_1(X_{1,i}) + \beta_2(X_{2,i}) + \epsilon_i $$

其中：
-   $Y_i$：第 $i$ 個群類的報名人數
-   $X_{1,i}$：對應的出生總人口
-   $X_{2,i}$：對應行業的平均總薪資
-   $\epsilon_i$：殘差項

**模型係數推論**：
-   **$\beta_1$ (人口推力係數)**：根據 3.1 節的分析，出生人口與總報名人數高度正相關。因此，我們預期 **$\beta_1$ 將會是顯著的正數**。這意味著，在其他條件不變下，出生人口越多，報考人數也越多。
-   **$\beta_2$ (薪資拉力係數)**：這是本研究的核心檢驗。根據散佈圖趨勢分析，薪資越高的行業，其對應的群類人數流失越緩慢（或人數越高）。因此，我們預期 **$\beta_2$ 也將會是顯著的正數**。

**結論**：若 $\beta_2$ 的檢定結果確實為顯著正數，即可做出本研究的最終結論：**在控制了少子化的人口衝擊後，產業的薪資水平對於學生的科系選擇，確實存在顯著且正向的影響力。** 這將強而有力地證明，技職體系的學生在面對未來時，是一位理性的「經濟人」。
