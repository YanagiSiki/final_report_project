# 二、 資料蒐集

本章詳細說明本研究資料來源、研究區間，以及為連結教育體系與勞動市場而進行的變數對照邏輯。

-   **資料來源**：
    1.  **技專校院入學測驗中心**：提供統測各群類歷年報名人數 (檔案: `tcte_registration_109_114.csv`)。
    2.  **行政院主計總處**：提供各行業別歷年薪資統計 (檔案: `salary_data_109_113.csv`)。
    3.  **內政部戶政司** (透過教育部統計處資料)：提供統測對應學年度之出生總人口數 (檔案: `tcte_birth_cohort_statistics_109_113.csv`)。

-   **研究對象與區間**：
    本研究主要分析民國 109 學年度至 113 學年度（部分資料延伸至 114 學年度）的台灣技職體系學生報考行為。研究對象為參與統測的學生，主要探討少子化與產業薪資對其科系選擇的影響。

-   **資料處理 (Mapping)**：
    為檢驗薪資對學生選擇的影響，本研究將教育體系的「統測群類」與勞動市場的「行業別」進行對照。此對照基於群類專業訓練與畢業後主要就業方向的直接關聯性。具體對照關係如下：
    -   **16 餐旅群** 對應 **住宿及餐飲業**
    -   **04 資電類 / 03 電機類** 對應 **出版影音及資通訊業**
    -   **09 商業管理群** 對應 **金融及保險業** (取高標)
    -   **01 機械群 / 02 動力機械群** 對應 **製造業**
    -   **06 土木與建築群** 對應 **營建工程業**
    -   **15 衛生與護理類** 對應 **醫療保健及社會工作服務業**
    -   **20 藝術群影視類** 對應 **藝術娛樂及休閒服務業**
    -   **05 化工群** 對應 **製造業** (化學材料製造)

    此外，本研究定義**人數變動率** = $(N_{113} - N_{109}) / N_{109}$，以標準化方式衡量各群類在研究期間的招生人數變化，作為學生選擇的結果指標。
