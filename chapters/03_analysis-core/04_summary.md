# 3.4 本章總結

本章透過一系列循序漸進的統計分析，旨在驗證在少子化的大背景下，行業薪資對技職科系報名人數是否扮演了關鍵的「拉力」角色。整個分析過程揭示了一個從初步觀察到最終統計檢定，結論發生轉變的重要歷程。

1.  **確立分析前提 (3.1 & 3.2 節)**：

    - 我們首先透過皮爾森相關分析，證實了出生人口數與統測總報名人數之間存在**高度顯著的正相關** (r = 0.97, p < 0.01)，確立了「少子化」是驅動人數下滑的根本性「推力」。
    - 接著，One-way ANOVA 分析結果表明，不同行業別之間的平均總薪資存在**極其顯著的差異** (F = 233.991, p < 0.001)，證明了「薪資」作為一個潛在的「拉力」因素，其本身在市場上具有足夠的區分度。

2.  **薪資的雙重效應 (3.3 節)**：
    - **穩定效應 (Stability Effect)**：分析顯示，高薪產業（如電資、醫護）在少子化浪潮中展現了顯著的韌性，其報名人數衰退幅度遠小於低薪產業（如餐旅）。絕對薪資水準成為了減緩生源流失的重要防線。
    - **擴張效應 (Expansion Effect)**：市佔率分析證實，學生確實會向高成長領域集中。雖然總人數在減少，但薪資成長率較高的科系，其市場佔有率呈現擴張趨勢。

**本章核心結論：**

本研究最關鍵的發現是：**薪資在少子化時代扮演了「穩定」與「擴張」的雙重角色。**

1.  **穩定生源**：高絕對薪資能有效降低學生流失的速度。
2.  **驅動重分配**：薪資成長則是驅動學生在不同科系間移動、重分配市場份額的關鍵動力。
3.  **理性選擇**：學生的選擇行為展現了高度的經濟理性，傾向於追求長期回報較高的領域。

因此，本章的結論重新定義了薪資的作用機制：**在人口退潮時，薪資高低與成長潛力共同決定了誰能留住並吸引最後的生源。**
