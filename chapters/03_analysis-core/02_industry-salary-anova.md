# 3.2 產業現實：檢視薪資結構差異 (Industry Reality: Examining Salary Disparities)

如第二章**圖 2-1** 的視覺化探索所示，不同行業的薪資水平存在明顯的差距。本節旨在透過變異數分析 (ANOVA)，從統計學上嚴謹地驗證此差異是否顯著，以確立「薪資」作為一個有效區分市場的變數。

---

### 3.2.1 分析方法

本節採用**單因子變異數分析 (One-way ANOVA)**。此方法用於檢定三個或以上獨立組群的平均數是否存在顯著差異。其核心思想是比較**組間變異 (Between-group variation)** 與**組內變異 (Within-group variation)**，若組間變異顯著大於組內變異，則可推斷各組的平均數不全相等。此比較的結果即為 F 統計量。

-   **相關公式 (F 統計量)**:
    ```math
    F = \frac{\text{MSB}}{\text{MSW}} = \frac{\sum_{j=1}^{k} n_j (\bar{Y}_j - \bar{Y})^2 / (k-1)}{\sum_{j=1}^{k} \sum_{i=1}^{n_j} (Y_{ij} - \bar{Y}_j)^2 / (N-k)}
    ```
-   **變數與參數說明**:
    -   **$Y_{ij}$**: 第 $j$ 個行業別（組）中的第 $i$ 個薪資觀察值。
    -   **$\bar{Y}_j$**: 第 $j$ 個行業別的平均薪資。
    -   **$\bar{Y}$**: 全體觀察值的總平均薪資。
    -   **$k$**: 行業別的總數。
    -   **$N$**: 薪資觀察值的總數。

-   **研究變數**:
    -   **因子 (Factor)**: **行業別** (`行業別`)，此為我們的分組變數。
    -   **應變數 (Dependent Variable)**: **總薪資** (`總薪資`)，此為我們要檢定其平均數差異的變數。

-   **研究假說**:
    -   **虛無假設 ($H_0$)**: 所有行業別的平均總薪資皆相等 ($\mu_1 = \mu_2 = \dots = \mu_k$)。
    -   **對立假設 ($H_1$)**: 至少有兩個行業別的平均總薪資不相等。

### 3.2.2 分析結果與討論

ANOVA 分析的結果摘要如下，此分析涵蓋了 109-113 年的所有薪資數據：

| F 統計量 (F-statistic) | p 值 (p-value) |
| :---: | :---: |
| **242.65** | **< 0.001** |

**結果判讀：**
F 統計量高達 242.65，其對應的 p 值遠小於 0.001 的顯著性水準。因此，我們有極充分的理由**拒絕虛無假設**，並接受對立假設。此結果強而有力地表明，**不同行業別之間的平均總薪資存在極其顯著的統計差異**。

### 3.2.3 小結

本節透過 ANOVA 分析，從統計上證實了勞動市場上存在顯著且穩固的**行業薪資壁壘**，或稱「貧富差距」。高薪行業與低薪行業的區隔並非暫時現象，而是一個長期存在的結構。這為我們的核心假說提供了關鍵支持：既然「錢景」的差異是如此的明確且持續，學生在做生涯規劃時，將此差異納入考量，便是一種理性的表現。有了這個前提，我們可以在下一節進一步探討，這道持續存在的薪資鴻溝，是否真的對應著學生選擇的消長。
